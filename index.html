Privacy Policy - USD to LKR Converter

Information Collection and Use

The USD to LKR Converter extension does not collect or store any personal information. We only process the currency values you input or select for conversion purposes.

API Usage

This extension uses the Exchange Rate API (exchangerate-api.com) to fetch current exchange rates. When you use the converter:

  • We send requests to exchangerate-api.com to get the latest exchange rates
  • No personal information is sent with these requests
  • Exchange rate data is cached locally in your browser for one hour to minimize API calls

Data Storage

All conversions are performed locally in your browser. We don't store or transmit your conversion history or any other usage data.

Third-Party Services

The only third-party service we use is the Exchange Rate API. You can review their privacy policy at https://www.exchangerate-api.com/terms.

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any changes by updating the date below.
